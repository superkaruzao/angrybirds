Cadastre-se agora!

Prepare-se para uma aventura de lançamento emocionante! Toque e segure o seu personagem para carregar o estilingue. Em seguida, arraste o dedo na tela para ajustar o ângulo e a força do lançamento. Solte o personagem para lançá-lo em direção aos porcos inimigos. Durante o vôo, use as habilidades especiais de cada personagem tocando na tela. O objetivo é derrubar todos os porcos para avançar para o próximo nível.

Derrubar todos os alvos inimigos em cada fase para avançar. À medida que progride, as fases se tornam cada vez mais desafiadoras. Você está pronto para essa emocionante aventura?

FAQ

Perguntas frequentes

Como faço para jogar?

Para jogar você deve criar uma conta de usuário, fazer um depósito financeiro e, em seguida, você terá acesso ao jogo.

Posso jogar sem depositar dinheiro?

Infelizmente, não. O depósito é necessário para acessar o jogo e participar das atividades.

Quais são as recompensas?

Ao completar o desafio você recebe uma % de ganho em cima do valor inserido naquela partida, além disso voce tem a chance de jogar a fase bônus, que multiplica muitas vezes o seu saldo.

Como faço para proteger minhas informações pessoais?

Nós implementamos medidas de segurança para proteger suas informações pessoais. Certifique-se de não compartilhar suas informações de conta com terceiros.

Posso jogar em dispositivos móveis?

Sim, o jogo é compatível com dispositivos móveis. Basta fazer clicar no botão "JOGAR" dentro do seu painel.

O que acontece se eu violar os Termos de Uso?

Se você violar os Termos de Uso, sua conta poderá ser suspensa ou encerrada a nosso critério.
